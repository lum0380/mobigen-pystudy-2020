## Using pip
-----

### pip 과 pip3
- 콘다 내에서 pip과 pip3중 어느 것을 써야 하는지에 대해 논의.
- 콘다 내에서 pip의 path를 살펴본 결과 다음과 같음
  - pip3 : /usr/bin/pip3 -> 패키지 설치 시 전역라이브러리path에 설치됨
  - pip : <conda_env>/bin/pip -> 패키지 설치 해당 환경에만 설치됨
